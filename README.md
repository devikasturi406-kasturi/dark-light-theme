🚀 About the Project
A lightweight, static web application designed to showcase custom CSS theme management and user-preference retention. This project focuses on a clean desktop user experience and smooth UI state manipulation.

🛠️ Key Features
Dynamic Theme Toggle: Smooth transitioning between dark and light modes using modular CSS variables.
State Persistence: Leverages browser localStorage to ensure the selected theme stays active on page refresh or after closing and reopening the tab.
Desktop Focused: Developed with a strict layout focus tailored exclusively for desktop displays.
